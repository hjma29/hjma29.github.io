    HPE Synergy Virtual Connect CLI

    Synergy Virtual Connect offers read-only CLI access. You need to use default "netop" operator account and configure the password to login.

    HPE Synergy Virtual Connect Read-only CLI manual

    This is the not-yet-published but sharable CLI manual from HPE engineering team.

    This page documents some screen capture and useful comments to make users easier to get started.

    SSH Access

    Step 1. Note down Synergy virtual connect mgmt IP address

    Step 2. Configure "netop" user password

    Step 3. SSH into VC mgmt IP address using netop account.

    $ ssh <virtual connect module mgmt IP address>
